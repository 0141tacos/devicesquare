# devicesquare
デバイスオタクがお気に入りのデバイスを記録し、共有・交流するためのSNS
## 概要
devicesquare は、ユーザーがお気に入りのデバイスを紹介し、他のユーザーと交流できるSNS風のWebアプリケーションです。
## 特徴
- お気に入りデバイスの投稿・画像アップロード
- 他ユーザーの投稿閲覧・コメント
- シンプルで直感的なUI
## 技術スタック
- Python (バックエンド)
- HTML/CSS (フロントエンド)
